# isomorphic-react-boilerplate

Boilerplate for quick start using 

- React
- React Router with Dynamic route import.
- React Helmet
- React Google Analytics
- Redux
- Styled components
- Code Splitting
- Server Side Rendering
- Long Term Asset Caching
- Jest
- Enzyme
- Webpack
- Hot Module Replacement
- Browser Sync
- Webpack Dashboard
- Friendly Errors
- ESLint
- Prettier
- Storybook

Under consideration

- Typescript
- TSLint

## Installation

`npm install`

## Run Unit Tests

`npm run test`

## Run Storybook Documentation

`npm run storybook`

## Run Development environment

`npm start`
